### Desktop (> 1024px)
- 2 columns for checkboxes
- Full width modals
- All details visible

### Tablet (768px - 1024px)
- 2 columns for checkboxes
- Slightly narrower modals
- Scrollable sections

### Mobile (< 768px)
- 1 column for checkboxes
- Full-screen modals
- Collapsible sections by default
